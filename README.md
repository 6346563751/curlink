
# Rxsum

Rxsum is a decentralized market bringing together groups and professionals.

## Deploy your own


Nodejs and npm (or yarn) are required to deploy the project.
```
git clone https://github.com/6346563751/rxsum.git
cd rxsum
npm install
```
For local development :
```
npm run dev
yarn dev
```
For build :
```
npm run build
yarn build
```
